h1. Seabass

A suite of Seabass-safe Rake tasks for deploying .NET applications

h1. Installation

To install the gem you just need to run

<pre>
gem install seabass
</pre>

h1. Example

<pre>
require 'rubygems'
require 'seabass'

robocopy do |r|
	r.files << "foo.txt"
	r.files << "bar.txt"
	r.directories << "c:/temp/source/baz"
	r.destination = "c:/temp/destination"
end
</pre>
